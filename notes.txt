To install yarn
    npm install yarn -g

To create a react Applications
    npx create-react-app <app_name>
    OR
    yarn create react-app <app_name>

To start the Application
    npm run start
    OR
    yarn start

To install axios
    yarn add axios
    OR
    npm install axios
